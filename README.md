####################################################
L.E.O. 
####################################################
Es un sistema de gestion de logisitica para ordenes de comida automatizado
con el fin de optimizar tiempo y energia en la gestion de ordenes.


####### CAUSA ###############
L.E.O surge de la necesidad de optimizar un proceso repetitivo sin necesidad de que una persona lo ejecute
Con el fin de agilizar el proceso de pedir el Menu del dia a un restuarante,
Comunicar la comida del dia a los usuarios y gestionar las ordenes de comida de manera automatica.
Es una tarea que se repite de Lunes a Viernes en el mismo horario y el mismo lugar.


####### SOLUCION ###############
Crear un asistente (L.E.O) que se encarge de gestionar todo el proceso de:
- Perdir el menu del dia al restuarante
- Notificarselo a los usuarios
- Gestionar las ordenes de manera automatica
- Enviar las ordenes al restaurante
